### API: Games

### Available Methods

* `boxscore`
  * Parameters:
    * `gameID`

* `content`
  * Parameters:
    * `gameID`

* `game`
  * Parameters:
    * `gameID`

* `games`
  * Parameters:
    * `date`
    * `startDate`
    * `endDate`
    * `gameType`

### Parameter Reference

  * `date`
    * Possible Values:
      * Date format: `yyyy-mm-dd`

  * `endDate`
    * Possible Values:
      * Date format: `yyyy-mm-dd`

  * `startDate`
    * Possible Values:
      * Date format: `yyyy-mm-dd`

  * `gameType`
    * Possible Values:
      * Preseason: `PR`
      * Regular season: `R`
      * Postseason: `P`
        * Note: defaults to current season
